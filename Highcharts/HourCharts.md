


How to set the data series
`data:`

```
className: Additional class name to the point
colorIndex: In styled mode return a `n` where is the index
dataLabels: Customize the points for the labels
description: Require the accessibility
events: Bunch of events that can do things
high: the maximum value for each data point (max)
low: the minimum value (min)
labelrank: the z-index for the labels
name: name of the point will be use for the label
x: The value of the point, for the X value is the timestamp in milliseconds
```
